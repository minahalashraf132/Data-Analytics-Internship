# Project 2 – Exploratory Data Analysis (EDA)

## Objective

Analyze sales data using Microsoft Excel to discover trends, patterns and business insights.

---

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Dashboard
- Slicers

---

## Dataset

Sales dataset containing:

- Products
- Orders
- Payment Methods
- Order Status
- Referral Sources

---

## Dashboard KPIs

- Total Sales
- Total Orders
- Average Order Value
- Average Quantity

---

## Charts

- Total Sales by Product
- Monthly Orders Trend
- Orders by Status
- Orders by Payment Method
- Orders by Referral Source

---

## Features

- Interactive Dashboard
- Pivot Charts
- Slicers
- Professional Layout

---

## Key Insights

- Product sales comparison
- Monthly order trends
- Distribution of order status
- Preferred payment methods
- Customer referral sources
